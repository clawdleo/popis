# 🥩 Popis

Shopping list app based on your Konzum receipt. Check off items as you pick them up.

## Features
- Categorized items (dairy, meat, produce, etc.)
- Checkboxes with progress tracking
- Quantities and amounts
- Persists state (localStorage)
- Move checked items to bottom
- Reset button
